This portfolio is built with Gatsby, React, and Tailwind.

## Quick start
1. **Install dependencies**

```
cd portfolio/
npm install
```

2. **Run local dev server**
```
npm run dev
```

3. **Ready for development!**

The site should now be running at http://localhost:8000

## Deployment to prod
```
npm run deploy
```
