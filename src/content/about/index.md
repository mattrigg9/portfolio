There are so many skilled programmers out in the world, but only a fraction truly appreciate the delicate relationship that a user has with their machine. This relationship fascinates me. This relationship is why I love to create.

A good interface should anticipate a users need before they even need to think about it, offering an experience that feels magical. I like to think of a well designed application like an airport. No matter where the user is, they should have enough context and spatial awareness to orient themselves and know what must be done next. Providing a user with this sense of control enables them to expertly navigate the application, no matter how big.

My name is Matt Rigg, and I create applications for humans.
