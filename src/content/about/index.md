## Why I build
There are so many skilled programmers out in the world, but only a fraction truly appreciate the delicate relationship that a user has with their machine. This relationship fascinates me. This relationship is why I love to design and develop applications.

A well-built application should anticipate a users need before they even need to think about it, offering an experience that feels magical. I like to think of a well designed application like an airport. No matter where the user is, they should have enough context and spatial awareness to orient themselves and know what must be done next. Providing a user with this sense of control enables them to expertly navigate the application, no matter the size of scope.

## What I build 
While my passions extend broadly across Front End Engineering, my specialties include writing scalable and modern Javascript, leveraging rendering frameworks like React and Vue.js. I have intimate experience building CSS frameworks that are semantic and thoughtful, enabling developers to build complex use cases on top of the foundation.

As a certified AWS Solutions Architect, I have extensive experience in the AWS ecosystem. This includes designing and building serverless microarchitecture that can scale to millions of users. 

Through all of this technical experience, my roots run deep but my motives are simple. I build applications that people love to use.
